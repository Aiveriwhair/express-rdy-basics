# express-rdy-basics

This is a basic express server setup with error handling, normalized responses and logging.

## License

[MIT](https://choosealicense.com/licenses/mit/)

## Installation

```bash
npm install
```

## Usage

### Build the project

```bash
npm run build
```

### Start the server

```bash
npm run start
```

### Start the server in watch mode

```bash
npm run dev
```

## Development

```bash
npm run dev
```
